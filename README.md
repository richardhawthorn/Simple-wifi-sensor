### Description

Designed to collect and report various sensor readings over wifi.

### Sensors

- Temperature
- Humidity
- Light
- Vibration
- IR Receiver

### Other Features

- ESP8266
- 3.3V operation
- Mini USB power connector
- FTDI programming port
- IR Transmitter
